# Titanic Data Analysis

## Project Overview

This project explores the famous Titanic dataset using Python and Pandas.

The goal was to practice:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Data Visualization

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset

Titanic - Machine Learning from Disaster

## Data Cleaning

- Removed Cabin column (77% missing values)
- Filled missing Age values using Median
- Filled missing Embarked values using Mode
- Created FamilySize column
- Checked for duplicate records

## Key Insights

### Survival Rate

- 38.4% survived
- 61.6% died

### Gender Analysis

- Female survival rate: 74.2%
- Male survival rate: 18.9%

### Passenger Class

- 1st Class: 63.0%
- 2nd Class: 47.3%
- 3rd Class: 24.2%

### Family Size

- Traveling with family: 50.6%
- Traveling alone: 30.4%

### Age Groups

- Child: 58.0%
- Teen: 41.1%
- Adult: 36.5%
- Senior: 26.9%

### Interesting Finding

- Women in 1st Class: 96.8% survival rate
- Men in 3rd Class: 13.5% survival rate
